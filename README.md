# MechCalc: Engineering Suite
An interactive dashboard for mechanical calculations.

## 🔗 [Live Website](https://adithravi.github.io/)

## ✨ Key Features
- **Real-time Stress Analysis:** Calculates stress and Safety Factor.
- **Material Presets:** Includes Steel, Aluminum, and Titanium.
- **UX Focused:** Color-coded safety alerts (Green = Safe, Red = Failure).

## 🛠️ Tech
- HTML, CSS, JavaScript (ES6)
- Built for Mechanical Engineers & Students
